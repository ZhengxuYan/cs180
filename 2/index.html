Project 2: Fun with Filters and Frequencies!

Jason Yan

This project explores various image processing techniques, including finite difference operators, Gaussian filters, image sharpening, hybrid images, and multiresolution blending.

Part 1.1: Finite Difference Operator

In this part, we implemented a finite difference operator to detect edges in images. The operator calculates the gradient in both x and y directions, allowing us to identify areas of rapid intensity change.

Part 1.2: Derivative of Gaussian (DoG) Filter

The Derivative of Gaussian (DoG) filter is a technique used in image processing for edge detection. It combines Gaussian smoothing with differentiation to highlight edges in images.
